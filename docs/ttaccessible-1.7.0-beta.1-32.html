v1.7.0-beta.1 (build 32) — 2026-06-20

This is a beta release, for testing a new feature before it ships to everyone. To receive beta updates, turn on Include beta versions in Preferences › General.

New: BearWare web login

  • Sign in with a BearWare account. You can now connect to servers that use BearWare web login (bearware.dk) without creating a separate account on each one. Set up your free BearWare account once in Preferences › BearWare, then turn on Use BearWare web login for any server that supports it.

This feature is brand new and looking for testers — feedback is very welcome via Help › Contact the Developer.

Install

If you have beta updates enabled, ttaccessible will install this update for you — no action needed.

Manual install:

  1. Download ttaccessible-1.7.0-beta.1-32.zip below.
  2. Unzip and drag ttaccessible.app into your /Applications folder, replacing the previous version.
  3. Double-click — no Gatekeeper warning thanks to notarization.
