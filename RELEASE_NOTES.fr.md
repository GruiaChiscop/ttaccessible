## v1.7.0-beta.10 (build 41) — 2026-07-07

Ceci est une version **bêta**, destinée à tester les nouveautés avant leur diffusion à tout le monde. Pour recevoir les mises à jour bêta, activez **Inclure les versions bêta** dans Préférences › Général.

### À la une
- **La connexion rapide est de retour.** Sur les Mac équipés de plusieurs périphériques audio, se connecter à un serveur était repassé à une quinzaine de secondes — cette version rend l'opération de nouveau quasi instantanée.

### Nouveautés

**Se connecter redevient rapide.** Une accélération du démarrage ajoutée il y a quelques bêtas ne se retrouvait en réalité pas dans les versions distribuées. Résultat : sur les configurations dotées de nombreux périphériques audio — interfaces, périphériques virtuels, casques —, l'ouverture d'une connexion pouvait se figer une dizaine de secondes avant que quoi que ce soit ne se passe, alors que les configurations plus simples fonctionnaient bien. C'est désormais corrigé à la racine : l'optimisation est intégrée à chaque version pour de bon, et la connexion est de nouveau quasi immédiate.

Rien d'autre ne change dans cette version — mêmes fonctionnalités et même audio que la bêta 9.

## Installation

Si vous avez activé les mises à jour bêta, tt-Accessible installera cette mise à jour pour vous — aucune action nécessaire.

Installation manuelle :

1. Téléchargez `ttaccessible-1.7.0-beta.10-41.zip` ci-dessous.
2. Décompressez l'archive et glissez `ttaccessible.app` dans votre dossier `/Applications`, en remplaçant la version précédente.
3. Double-cliquez — aucun avertissement Gatekeeper grâce à la notarisation.
